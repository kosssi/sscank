Permet de facilement aligner des éléments

Pour l'alignement de texte

Du texte à droite (txtr)
Du texte à gauche (txtl)
Du texte centré (txtc), en dessus du texte caché (txth)
Du texte caché

Pour l'alignement de block (box)

Dans un block (.box), il est possible d'avoir des blocks avec les classes suivantes :

  • top : pour mettre un élément en haut
  • bottom : pour mettre un élément en bas
  • left : pour mettre un élément à gauche
  • center : pour mettre un élément centré horizontalement
  • right : pour mettre un élément à droite

Pour des blocks HTML5 (box5)

La position des blocks à l'aide de flexbox permet de créer des conteneurs positionnable :

  • top (default) : pour mettre un élément en haut
  • middle : pour mettre un élément centré verticalement
  • bottom : pour mettre un élément en bas
  • left (default) : pour mettre un élément à gauche
  • center : pour mettre un élément centré horizontalement
  • right : pour mettre un élément à droite
  • horizontal (default) : pour mettre les éléments les un à la suite des autres
  • vertical : pour mettre les éléments les un au dessus des autres

Attention les flexbox sont compatible avec les navigateurs récents.
